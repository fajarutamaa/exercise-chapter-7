## Installation

Clone the project from GitHub repository:

      git clone https://github.com/fajarutamaa/exercise-chapter-7.git

Change Directory:

      cd exercise-chapter-7

Install all package dependencies:

      npm install

Compile and hot-reload for development:

      npm run dev
